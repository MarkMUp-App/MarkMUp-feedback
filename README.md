# MarkMUp — Feedback & Support

Welcome! This is the public issue tracker for the Markify app.

## What is MarkMUp?
Convert any file to Markdown — PDF, DOCX, XLSX, PPTX, CSV, 
HTML, JSON, XML, TXT and more. Works 100% on your device. 
No cloud. No AI. Your files never leave your phone.

## Report a bug or request a feature
